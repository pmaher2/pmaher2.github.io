Functions

Functions are essential when creating a program. They can be used to execute code when data is given.

One of the most basic functions is the main function:

  • public static void main(String[] args)
  • {
  •  
  • }

This is the standard format for the main function inside a java class and is used to run other functions.

We can add custom functions or other lines of code inside of this to run them.

  • public static void main(String[] args)
  • {
  •  System.out.println("Hello World!");
  • }

The line of code added above will print the following to the console:

 Hello World!

When possible, you should put code inside of functions other than the main function to stay organized. This is one of many concepts used in Object Oriented Programming.

Note: Functions should be declared outside of the main function but invoked inside of the main function.

What can be done with functions?

Functions can be called inside of other functions. For example:

  • public static void doSomething()
  • {
  •   otherFunction();
  •   anotherFunction();
  •   ...
  • }

A function that does not have a return value is known as a void function.

Additionally, functions can have return values which can return a value from a mathematical operation or a location in memory. The most common forms of return functions are:

  • boolean which returns true or false
  • integer which returns a whole number
  • double which returns a float (number with decimal)
  • public static int getAverage()
  • {
  •   return (1+2+3)/3;
  • }
