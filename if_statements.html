Conditional Statements

A conditional statement is a piece of code that is only ran if a certain condition is met.

If statements

If statements are a type of conditional statement in which numbers, booleans, or other objects can be compared against each other or against another value.

To compare two booleans:

  • boolean var1 = true;
  • boolean var2 = false;
  • if(var1 == var2)
  •    System.out.println("equal");
  • if(var1 != var2)
  •    System.out.println("not equal");

In the first if statement var1 is being compared to var2 to test if they are equal. Since var1 is true and var2 is false meaning they are not equal, this statement is ignored and we move on to the next line.

The next line is another if statement. Both variables are being tested to see if they are not equal. The variables are not equal to each other which results in the statement being true and therefore it is printed to the console:

  • not equal

Else statements

Else statements are similar in function to if statements except they are only executed if the first if statement is false.

For example:

  • int num = 0;
  • if(num < 0)
  •    System.out.println("Number is negative");
  • else
  •    System.out.println("Number is positive");

Would print the following to console:

  • Number is positive

In some cases else should be used instead of two if statements.

If both if statements are true both conditions could be met, resulting in two actions being performed. To avoid this use an else statement so only one action is performed.

Else if statements

In some situations else if statements are ideal.

In the code below, assume grade has been initialized

  • double grade;
  • if(grade >= 90)
  •    System.out.println("letter grade is A");
  • else if(grade >= 80)
  •    System.out.println("letter grade is B");
  • else if(grade >= 70)
  •    System.out.println("letter grade is C");
  • else if(grade >= 60)
  •    System.out.println("letter grade is D");
  • else
  •    System.out.println("letter grade is E");

The first statement in the code above has to be an if statement.

The last statement is an else statement because it can only come directly after an if or an else if statement.

If the code were to have all of the statements as if statements instead of else if statements, the code would print out messages saying the grade is multiple letters instead of just one
