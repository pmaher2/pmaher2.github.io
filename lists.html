Introduction to lists

Lists are often used to store large amounts of data in a form that can be accessed and used again later

Most lists are initialized in a way that is similar to normal variables but with a set of brackets immediately after the object type. Like variables, lists also must follow the standard naming rules

Standard array lists have a fixed length that must be set when the array is initialized.

In the following example, an array of type int is initialized to a new int array.

  • int[] example = new int[5];

Using toString()

By default each value in this list will be set to 0. To print this to console as a readable string, use the predefined toString function:

  • System.out.println(Arrays.toString(example));

If the toString function is not used, the location of the int array will be printed to console rather than the values

The output should look like this:

  • [0, 0, 0, 0, 0]

To add values to the array:

  • example[2] = 5;

This will print:

  • [0, 0, 5, 0, 0,]

to the console

Using .length()

To get the length of any standard list in java, use the .length() function. This is a predefined function that returns the length of the list.

To do this, use the following code:

  • System.out.println(example.length());

This will print out the length to console

The .length() function is often used in loops to index through a list of values.

ArrayLists

An ArrayList is a type of list in which the size can vary. Like normal lists, ArrayLists must have a specified object type when defined.

ArrayLists do not use int and double but rather use Integer and Double. This is because ArrayLists must be set to an Object rather than a primitive type. ArrayLists can have an object of type int[] or double[] since arrays are not primitive types.

Learn more about the difference between primitive types and objects here.

To initialize an ArrayList use the following syntax:

  • ArrayList<Double> ar = new ArrayList<Double>();

Using .size()

To return the size of an ArrayList, invoke the .size() function on the array.

  • System.out.println(ar.size());

Adding values to an ArrayList

Unlike normal lists, ArrayLists use the .add() function to add a value to the it.

  • ArrayList<Double> ar = new ArrayList<Double>();
  • ar.add(1.0);
  • ar.add(1.5);
  • ar.add(2.0);

When printed to console the output will look like:

  • [1.0, 1.5, 2.0]

Setting values in an ArrayList

Similarly to standard lists, ArrayLists use an index to set the value at a location in the list. With ArrayLists the .set() function is used.

The .set() function accepts two parameters. The first parameter is the index and the second parameter is the element to add to the ArrayList.

Example:

  • ArrayList<String> ar = new ArrayList<String>();
  • ar.add("fruit 1");
  • ar.add("fruit 2");
  • ar.add("fruit 3");

Output:

  • [fruit 1, fruit 2, fruit 3]

To change the values of the list use the .set() function:

  • ar.set(0, "apple");
  • ar.set(1, "orange");
  • ar.set(2, "banana");

Once printed to console the output will look like:

  • [apple, orange, banana]
