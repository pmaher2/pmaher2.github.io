Common Variable Types

The most commonly used variable types in java are:

  • a boolean which can be set to true or false
  • an integer which can be set to any whole number
  • a double which can be set to any decimal value
  • a string which contains text
  • a char which stores a single character

Variable names have a few requirements to be valid:

  • The first character must be either a letter, dollar sign, or underscore.
  • Cannot be a keyword or reserved word such as class, if, for, null, void, etc.
  • A variable cannot have the same name as one already defined

In addition to this, variable names should be short but descriptive. However, this is not a requirement.

To create a variable:

  • boolean example1 = true;
  • int example2 = 4;
  • double example3 = 3.14;

Strings

Strings are a type of variable that can store text. Strings can contain numbers but they are still treated as text rather than a numerical value.

All strings must start and end with quotation marks

The following examples are valid strings:

  • String text = "This is a String";
  • String text2 = "This is also String";
  • String text3 = "Hello 123";

Concatenation

Concatenation is when strings are combined to form another string

  • String firstName = "Patrick ";
  • String lastName = "Maher";
  • String fullName = firstName + lastName;
  • System.out.println(fullName);

The resulting string is printed to the console:

  • Patrick Maher

Escape Sequence Characters

Escape characters can be used to insert tabs, move text to the next line and more.

Escape Sequence Description
\t tab
\b backspace
\n new line
\r carriage return
\f form feed
\' apostrophe
\" quotation mark
\\ backslash

For example:

  • System.out.println("Hello" + \nWorld!);

The resulting output:

  • Hello
  • World!

Using \t instead:

  • System.out.println("Hello" + \tWorld!);

Would print the following:

  • Hello    World!

Boolean values

Boolean values are often used in logical expressions to compare multiple values.

Boolean values are also used in if statements to determine whether or not an action should be performed.

  • boolean A = true;
  • boolean B = false;
  • System.out.println(A && B);

In the example above, A is true, B is false, and they are being compared using the and operator.

Since we are comparing true and false, false will be printed to the console

If we used or instead, true would be printed.

Integers and Doubles

Integers are useful for programs that utilize whole numbers

For example, if we want to make a program that sets a variable to the amount of students in classroom we would use integers and not doubles

Doubles should be used when we are dealing with data that can be a fractional part.

Examples of data that use a double variable:

  • Money
  • Test grades
  • Fractions

Examples of data that use an integer variable:

  • Number of students in a class
  • Cars in a parking lot
  • Population of a city

Casting

A double always has a decimal point in it, but it can be initialized to a whole number. If this happens, Eclipse will automatically cast the whole number to type double.

  • int num = 0;
  • System.out.println(num);

The number 0 is cast from int to double when initialized, resulting in the following output to console:

  • 0.0

An integer can only be set to whole numbers. Eclipse will not cast doubles to integers automatically.

Despite this, doubles can still be cast to integers but you must specifiy this

To cast from type int to type double we can use the following syntax:

  • double num1 = 4.8;
  • int num2 = (int) num1;

Doubles and integers are not the only objects in java that can be cast to another object.
